ProjectManager

ProjectManager — это веб-приложение для управления проектами и сотрудниками.
Система позволяет создавать записи о проектах, добавлять и удалять сотрудников, просматривать и редактировать весь список проектов.

Технологии
- ASP.NET Core MVC / Web API
- Entity Framework Core + SQLite
- AutoMapper
- Bootstrap + jQuery (простая фронтенд-логика, AJAX для поиска сотрудников)
